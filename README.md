# Student Manager (React + MUI)

This app implements the `StudentManager` component using React Hooks (`useState`) and Material UI components.

## Features
- Initial students array with 5 objects (`id`, `name`, `course`)
- Add new student with input boxes
- Delete student instantly without refresh
- Empty-state message: `No students available`
- Basic styling with responsive layout and animations

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
